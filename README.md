# Capwatch

Watch your cryptoportfolio in a console

![Basic Fund](http://i.imgur.com/BEu0KeU.png)

## Installation

    $ gem install capwatch

```bash
cat <<EOT > ~/.capwatch
{
  "name": "Basic Fund",
  "symbols": {
    "MAID": 25452.47,
    "GAME": 22253.51,
    "ANS": 3826.53,
    "FCT": 525.67875,
    "SC": 4152770,
    "DCR": 453.22,
    "BTC": 8.219,
    "ETH": 166.198,
    "KMD": 19056.20,
    "LSK": 5071.42
  }
}
EOT
```

    $ capwatch
